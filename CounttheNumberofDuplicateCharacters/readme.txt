Count the Number of Duplicate Characters
Create a function that takes a string and returns the number of dublicate characters.

Examples
duplicateCount("abcde") ➞ 0

duplicateCount("aabbcde") ➞ 2

duplicateCount("Indivisibilities") ➞ 1

duplicateCount("Aa") ➞ 0
// Case sensitive
Notes
Duplicate characters are case sensitive.
